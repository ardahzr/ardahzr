# Hi, I'm Arda Hazar

Computer engineering student and developer based in Istanbul. I build practical software across AI, desktop apps, web platforms, and computer vision.

I like turning course projects, experiments, and product ideas into usable applications with clear interfaces and real workflows.

## Focus Areas

- AI-powered tools and local LLM applications
- Full-stack web platforms for education and operations
- Computer vision experiments with YOLO and deep learning
- Desktop apps with Python, .NET, Tauri, Rust, and TypeScript
- Internet Computer and Motoko projects
