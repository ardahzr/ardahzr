## About

I enjoy building software that feels useful, not just experimental. My projects usually sit somewhere between AI, education platforms, desktop tooling, and computer vision.

Right now I am especially interested in local AI systems, clean full-stack workflows, and turning technical ideas into products people can actually use.

## Selected Work

| Project | Description | Stack |
| --- | --- | --- |
| [Licord](https://github.com/ardahzr/Licord) | Native Discord-inspired voice and group chat app focused on speed and a lighter desktop experience. | Tauri, Rust, TypeScript |
| [ACU AI Chatbot](https://github.com/ardahzr/acibadem-ai-chat) | Local AI chatbot for Acibadem University, built around an open-source LLM workflow. | Django, PostgreSQL, Ollama |
| [Program Outcome Manager](https://github.com/ardahzr/Learning-and-Program-Outcome-Management) | Academic outcome management app for calculating learning and program outcome scores from course grades. | Vue, Python |
| [LMP Music Player](https://github.com/ardahzr/linux-opus-audio-platform-with-youtube-spotify) | Linux desktop music player with local library playback and YouTube download support. | .NET, GTK, C# |
| [MiniOS](https://github.com/ardahzr/MiniOS) | Educational desktop operating system simulation with windows, taskbar, apps, and OS-course concepts. | Python, PySimpleGUI |
| [Scoliosis Classification](https://github.com/ardahzr/Scoliosis-Classification) | YOLOv8x-Pose based scoliosis detection experiment for analyzing spinal curvature. | Python, YOLOv8 |
